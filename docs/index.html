Sinter

AI creative coding studio for sketches, shaders, music systems, video compositions, and iterative generative workflows.

GitHub Repository Quick Start AI Discovery

What Sinter Is

A model-agnostic workbench where natural language becomes creative code, visible previews, and repeatable operator receipts.

Studio

Chat-first GUI with prompt, preview, revision, and receipts in one operator surface.

Creative Domains

Generate p5.js, GLSL, Three.js, SVG, Hydra, Strudel, Tone.js, Revideo, HTML, ASCII, kinetic pieces, and text.

Provider Flexibility

Works with OpenAI-compatible APIs, MiniMax, GLM, Anthropic-style providers, Ollama, and LM Studio.

Operator Proof

CLI, TUI, proof scripts, preview contracts, and audit receipts keep creative output inspectable.

Start Here

pnpm install
pnpm build
sinter --prompt "Create a luminous blue-green particle garden"
pnpm gui

Use pnpm tui for the Bubble Tea operator cockpit and sinter --configure to configure providers.

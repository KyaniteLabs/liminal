# Positioning

_Last updated: 2026-04-09_

## Category
**Creative-coding copilot**

## One-line pitch
Liminal is a creative-coding copilot for generating, previewing, and iterating code-native visuals across p5, shaders, Three, Hydra, Strudel, and related visual domains.

## What it is not
- not just another AI art app
- not just a text-to-video tool
- not just a code generator
- not an enterprise AI governance platform

## Core problem
Most generative tools produce flattened outputs or generic code snippets. Creative coders, design engineers, and live-visual artists want editable systems they can refine, remix, and perform with.

## Value proposition
This product helps users move from prompt → code → preview → revision in a way that preserves creative control.

## Primary audiences
### 1. Creative technologists
People building generative visuals, interactive sketches, and motion experiments.

### 2. Design engineers
People who want fast visual prototyping in code-native media.

### 3. Live visual artists / VJs
People who need generative visuals, audio-reactive systems, and rapid iteration.

### 4. Motion/interactive studios
Small teams prototyping concept visuals and expressive systems quickly.

## Strongest use cases
- generate p5 / shader / Three sketches from prompts
- rapidly explore aesthetic variants in code
- create motion/visual prototypes for pitches and experiments
- build live visual ideas faster than starting from a blank file

## Why this wedge is strong
The repo already reflects a multi-domain creative engine, not a single-medium tool. The clearest commercial angle is not “AI makes art” — it is “AI helps you work like a creative coder.”

## Messaging pillars
### Code-native
Outputs are editable systems, not just static deliverables.

### Multi-domain
The tool spans multiple creative coding environments.

### Iterative
The product is built around revision, evaluation, and refinement.

### Performable
The output can support visuals, motion, and live experimentation.

## Positioning statement
For creative technologists, design engineers, and live-visual artists who want editable generative systems instead of one-shot media, this product is a creative-coding copilot that generates and iterates code-native visuals across multiple visual domains.

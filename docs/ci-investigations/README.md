# CI Investigation Packets

Use this directory for one-file CI handoffs.

Operator rule: Simon should only need the packet path and the suggested agent.

Digest format:

```md
CI packet ready: docs/ci-investigations/YYYY-MM-DD-run-RUNID.md
Give to: <agent lane>
```

Packet rule: put the evidence, repro command, likely cause, and definition of done inside the file. Do not ask Simon to copy several blocks into another agent.

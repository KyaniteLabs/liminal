## Brand criteria
We want a name that is:
- distinctive
- easy to say and spell
- ownable in software/creative-tooling markets
- compatible with developer + artist audiences
- broad enough for visuals, motion, code, and live performance

## Winner: Refract

**Product narrative:** Light refracts when it passes through a medium, bending and splitting into new patterns. Code refracts when it passes through the harness — bending into visual form, splitting into generative patterns, creating something new.

**Launch plan:**
- Register refract.dev + refract.io
- Secure @refract on GitHub, Bluesky
- "Refract + creative coding" has near-zero search competition
- Trademark intent-to-use: Class 009 (software), Class 042 (SaaS)

**Logo concept:** Angular refraction — a single beam splitting into colored paths. Minimalist, geometric, iconic at favicon size.

## Backup: Facet

**Pivot trigger:** If ALL of these conditions are met:
- Facet.ai is acquired/pivots/becomes inactive
- refract.dev/refract.io unavailable or exceed $500
- "refract creative coding" not top-3 organic within 6 months
